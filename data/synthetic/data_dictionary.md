# Data Dictionary (synthetic)
- `participant_id` (str): pseudo ID
- `condition_seq` (str): "system-first" | "human-first"
- `condition_acc` (str): "claim" | "modify"
- `deference_score` (0–1): higher = more deference to system
- `voice_score` (0–1): higher = more participant voice
- `time_seconds` (int): call completion time
